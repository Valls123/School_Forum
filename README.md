# School_Forum
# 🎓 Школьный Форум

> Анонимный форум для школьников с системой кланов, секретным чатом и рейтингами.  
> Создан специально для Яндекс Лицея © 2026 Кирилл Эдуардович Шпаков

---

## 🚀 Стек технологий

| Слой | Технология |
|------|-----------|
| Backend | Python 3.13, Flask |
| База данных | SQLite + SQLAlchemy ORM |
| Авторизация | Flask-Login |
| Realtime | Flask-SocketIO + Eventlet (WebSocket) |
| Frontend | Jinja2, Tailwind CSS, Font Awesome |
| Архитектура | Blueprint-модульная структура |

## ✨ Возможности

### 👤 Пользователи
- Регистрация с валидацией ника (только латиница, начинается с буквы, 3–20 символов)
- Аватар и баннер профиля с загрузкой изображений
- Биография профиля
- Статус онлайн (активен последние 5 минут)
- Система верификации профиля (администрацией)
- Место в топе участников на странице профиля
- Подписки и подписчики

### 📝 Посты
- Создание постов с поддержкой абзацев
- Система голосования (лайки / дизлайки)
- Три режима сортировки: **Новые**, **Подписки**, **Фавориты** (рейтинг ≥ 1)
- Пагинация — 15 постов с кнопкой «Показать ещё» (AJAX, без перезагрузки)
- Скрытие постов на модерацию
- Открытие поста на отдельной странице

### 💬 Комментарии
- Комментарии к каждому посту
- Лайки и дизлайки к комментариям
- Закрепление комментариев (глава клана, модератор, администратор)
- Удаление комментариев (автор или модератор/администратор)
- Переход в профиль автора комментария

### 🔒 Секретный чат
- Доступ только для верифицированных участников и администраторов
- Realtime-обмен сообщениями через WebSocket
- Плавная анимация появления сообщений
- Закрепление сообщений
- Удаление сообщений модераторами

### 🛡️ Кланы
- Создание клана только верифицированными пользователями
- Три режима вступления: **Открытый**, **Ограниченный** (заявки/приглашения), **Закрытый**
- Роли внутри клана: **Глава**, **Страж**, **Участник**
- Публикация постов от лица клана (Глава и Страж)
- Голосование за посты клана
- Рейтинг клана на основе голосов за посты
- Верификация клана администрацией/модератором
- Аватар и баннер клана
- Система заявок и приглашений
- Кик участников (Глава клана или администратор сайта)
- Топ кланов по рейтингу

### 📰 Новости
- Раздел официальных новостей сайта
- Публиковать могут только администраторы
- Пользователи ставят лайки и дизлайки
- Информационный блок для обычных пользователей

### 🏆 Рейтинги и топы
- Топ-5 участников на главной странице (кеш 5 минут)
- Рейтинг пользователя — сумма голосов за все его посты (SQL-запрос)
- Топ кланов по рейтингу постов

### 🔧 Администрирование
- Панель управления пользователями
- Выдача ролей: **Администратор**, **Модератор**, **Участник**
- Верификация профилей
- Временная блокировка (30 мин / 6 ч / 1 д / 7 д / 30 д / произвольно)
- Постоянная блокировка
- Удаление профиля (каскадно удаляет посты, голоса)
- Доступ к секретному чату (флаг «секретный агент»)
- Модерация постов (скрытие / восстановление / удаление)
- Верификация и удаление кланов
- Кик участников из любого клана

## ⚙️ Установка и запуск

```bash
# 1. Клонировать репозиторий
git clone https://github.com/Valls123/School_Forum
cd ComputerGarden

# 2. Установить зависимости
pip install -r requirements.txt

# 3. Создать базу данных
flask shell
>>> from models import db
>>> db.create_all()
>>> exit()

# 4. Запустить сервер
python app.py
```

Открыть в браузере: [http://127.0.0.1:5000](http://127.0.0.1:5000)

> Первый зарегистрированный пользователь автоматически получает роль **Администратора**.

## 🔐 Безопасность

- Пароли хешируются через `pbkdf2:sha256` (Werkzeug)
- Проверка прав доступа на каждом маршруте (`@login_required`, проверка роли)
- Защита от CSRF через Flask-форм подтверждения
- Валидация никнейма по регулярному выражению
- Автоматический выход заблокированных пользователей при каждом запросе
